# Dev Logs Directory

This directory contains individual dev log entries for DIG-345 Radical Software.

## Naming Convention

Dev log files should follow this naming pattern:
```
YYYY-MM-DD-topic-or-title.md
```

For example:
- `2026-01-28-getting-started.md`
- `2026-02-05-javascript-basics.md`
- `2026-02-12-web-apis.md`

## Organization

Entries are organized chronologically by filename. The date prefix ensures they appear in order.

## Creating a New Entry

1. Copy the template from the root directory:
   ```bash
   cp ../TEMPLATE.md YYYY-MM-DD-your-topic.md
   ```

2. Update the date, topic, and fill in the sections

3. Add any images or media to the `/assets/` directory
